This group project involved building a machine learning analysis pipeline to predict diseases from patient-reported symptoms across 41 disease classes and 132 binary symptom features. The dataset was open source and obtained from Kaggle.

My contributions covered both the analysis work and the project presentation delivery.

On the technical side, I implemented and evaluated two models: Bernoulli Naive Bayes and Decision Tree. The Bernoulli Naive Bayes model was a deliberate choice for this dataset given its binary feature space, and it achieved 100% test accuracy despite violating the conditional independence assumption across 5 identified correlated symptom pairs (e.g., yellowish_skin and yellowing_of_eyes). The Decision Tree model reached 97.62% test accuracy, with the single misclassification being a Fungal infection case — the same ambiguous record that tripped up every other imperfect model. I also performed exploratory data analysis including symptom prevalence, correlation analysis, feature variance screening (which identified fluid_overload as a zero-variance feature), and disease-symptom association heatmaps. This produced 12 outputs across EDA charts, confusion matrices, feature importance rankings, and a model comparison summary.

Beyond the modelling work, I collaborated with Deeqa to prepare the project showcase presentation. Together we built the presentation slides and structured the narrative around our key finding: that simpler models matched or outperformed complex ones on this synthetic dataset, and that the results, while strong, cannot be generalized to clinical practice.

Given additional time, I would pursue two directions. First, testing all models on a real clinical dataset to understand how performance degrades under noisy, incomplete, and co-morbid conditions — the current synthetic results are encouraging but not clinically actionable. Second, I would build a simple interactive symptom checker as a proof-of-concept clinical tool, allowing users to input symptoms and receive ranked diagnoses with confidence scores, using Bernoulli Naive Bayes as the backend given its speed, interpretability, and native probability output.
